Portfolio of ANCELLY Eddy - Network & Telecommunication

This portfolio showcases projects and practical works completed by Eddy Ancelly in the field of networking and telecommunications for the academic years 2022 to 2025. It includes detailed reports and practical exercises highlighting my skills and achievements in this domain.

R301 OSPF Report

The main objective of this practical was to configure VLANs, enable inter-VLAN routing for communication between segmented networks, and deploy ACLs to control access between them. Packet Tracer was used for this implementation.

TP_MPLS

This report from TP R3.02 details the implementation and analysis of MPLS routing, covering initial configuration, connectivity tests, and performance evaluation. It also reviews MPLS label distribution, load balancing efficiency, and transmission delays, highlighting how MPLS enhances packet management and switching in networks.

IPv6 Practical

The goal of this practical was to build a virtual network using IPv6 protocol and explore IPv6 routing concepts. A Debian virtual machine with Netkit, Wireshark, SSH, and IPv6calc was used for the exercises.

SAE Transmission System

This SAE project aimed to design a three-character transmission system, including a transmitter, receiver, and frame, using the ADALM-Pluto through Simulink in Matlab.

TP: Web Server Security - TLS & High Availability

This report outlines the installation and configuration of a secure web server, focusing on high security standards and adherence to DIC-P criteria, which are essential for securing information systems.

R3.17 Advanced Virtualization

This practical explores the installation of VMware Pro for creating virtual environments, deploying a Windows 2022 server as a virtual machine, and illustrating the importance of virtualization in server management. It also covers deploying an ESXi virtual image via OVA format.

TP1 DNS Server

This report explains the installation and configuration of a DNS server.

R204 Telephony TD

This task focused on telephony concepts and their practical applications.

SAE Cyber Blue Wave

This report, created during the Cyber Blue Wave SAE, documents the technical analysis of the network infrastructure.
